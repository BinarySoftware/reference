parse
This method parses the ID producing a Zone.
Example
Get Central European Time.
Zone.parse "CET"
Example
Get Moscow time.
Zone.parse "Europe/Moscow"
Example
Get time zone -06:00.
Zone.parse "-06:00"
Example
Get custom offset +03:02:01 of 3 hours 2 minutes an 1 second.
Zone.parse "+03:02:01"
